So… About this server

TL;DR: Whitelisted vanilla Minecraft server, small number of people, lots of fiddleing in the background.

Server

The Minecraft server is accessible via wurstmannberg.de or mc.wurstmannberg.de – We haven't figured out domain stuff yet.

Our Minecraft server is currently running on a basic vServer. Nothing fancy, 1-2GB RAM, a single CPU core, 50Gigs of space. As more people join the server, we probably need to upgrade sooner or later, but oh well. Turns out decent vServers suitable for more than just fancy webhosting aren't as easy to find. Admittedly we haven't looked too hard as of now.

Before that we were using CubedHost as our server host, and… Well, Everything worked fine (except for one 24h downtime we never found out the reason for), but since we only had access via FTP and Multicraft, we soon ran out of stuff to tinker with. And besided, easy and fast snapshot updates, automated backups to different machines and all that aren't really fun to do without shell/ssh access.

Even before we tried CubedHost, the server was set up on a spare Laptop in Jemus42's apartment. Back then only Jemus42 and BenemitC where playing (needing a way to continue their multiplayer map they started with… less physical distance), so that actually worked. Kind of.So yeah, I guess you could say we've come a long way since then.

Map and Version

Our current map was first generated sometime in january '13, therefore the core of the map is Minecraft v1.3 terrain generation. We soon started using the early v1.4 snapshots and have been using the latest snapshot (if one is available) ever since. So expect occasional downtimes due to server upgrades and sometimes impending debugging session if the snapshot was a little funky. Another side effect of the snapshot use can be chunks that are generated with experimental settings, so when we upgrade to the v1.7 snapshots (bringing new terrain generation) it might be we get the occasional chunk with features that don't end up in the final release of v1.7. If that's a good thing or a bad thing, we'll find out.

Another thing concerning terrain generation and version differences: If you're looking for Nether Quartz, you may need to travel a little. But don't worry too much, we have a nether hub set up which allows you to safely (kind of) continue some tunnels inside the nether roof to get to newly generated nether terrain. As far as horses are concerned, you may be interested in L3viathan and Jemus42's fahrvergnuegen ranch in the south of the map, where the first horses on the server were found and are now kept for breeding, testing, experimentation and other words that sound morally ambiguous.
